주요 발견사항

  • Project Compass is a Europol European Counter Terrorism Centre (ECTC) project active since January 2025, ongoing as of the captured publication.
  • 28 country partners enumerated: Australia, Austria, Belgium, Canada (RCMP), Croatia, Czechia, Denmark, Finland, France, Germany, Greece, Hungary, Iceland, Ireland, Italy, Luxembourg, Malta, Netherlands, New Zealand, Norway, Poland, Portugal, Romania, Slovakia, Slovenia, Spain, Sweden, Switzerland, UK (UK CT Policing, NCA), US (FBI, HSI).
  • Target: 'The Com' (Community) — nihilistic extremist transnational virtual network (TVN) with three subgroups: Cyber Com (network intrusions / ransomware), Offline Com (terrorism / property damage), (S)extortion Com (CSAM extortion of minors).
  • Notable subgroup: 764 — active since 2021, decentralised online group recruiting and grooming minors for violence + CSAM production + blackmail.
  • Results snapshot: 4 victims safeguarded; 30 perpetrators arrested (contributed to AP Dolphin); 62 victims identified or partially identified; 179 perpetrators identified or partially identified; 9 joint awareness-raising activities.
  • Activities include data sprints where invited countries present ongoing cases; assistance to CT, CSAM, and organised-crime unit investigations; preventative-measure advice.

Source Summary

Europol Operations / Project Compass page documenting an ongoing 28-country Europol-coordinated effort, led by the European Counter Terrorism Centre (ECTC), targeting "The Com" — a nihilistic extremist transnational virtual network targeting minors. Active since January 2025; results snapshot (as of capture) include 30 arrests, 4 victims safeguarded, and 179 identified perpetrators across the partner countries.

IC Pattern

This release establishes a Europol ECTC-coordinated 28-country counter-extremism + CSAM cooperation framework that the wiki has not previously documented. Notable features:

  • Europol European Counter Terrorism Centre (ECTC) as the coordination authority — distinct from EC3 (cybercrime) or EFECC (financial crimes).
  • Five Eyes (5) + EU Member States (22) + EFTA partners (Iceland, Norway, Switzerland) in a single project framework — the wiki's largest enumerated single-project country roster outside of INTERPOL operations.
  • The Com / 764 as a subject — a transnational virtual network combining cyber intrusions, terrorism, and CSAM extortion of minors.
  • AP Dolphin referenced as a connected analytical project / Europol Analysis Project that received contribution from Project Compass arrests.
  • "Data sprints" as a discrete IC mechanism class — invited countries present their ongoing cases at Europol-organised meetings to coordinate investigations and identify cross-border links.

Wiki Cross-Corpus Relevance

This project complements: - Operation Stream / Kidflix CSAM platform takedown (March 2025) — single-platform CSAM takedown - Operation Cronos / LockBit — ransomware takedown (Cyber Com would structurally correspond) - LabHost PhaaS takedown (April 2024) — phishing-platform takedown

Project Compass adds the wiki's first record of a Europol ECTC-coordinated long-running project (vs. single-action takedown) targeting a transnational virtual network.
