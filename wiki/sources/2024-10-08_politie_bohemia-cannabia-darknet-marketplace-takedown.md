## What the source establishes

### Operational facts (confirmed across all sources)

| Fact | Value |
|---|---|
| Marketplace names | Bohemia and Cannabia (dual / sister markets, common admin) |
| Investigation start | Late 2022 |
| Marketplace shutdown | End of 2023 (alleged exit scam by operators) |
| First admin arrest | 2024-06-27, Schiphol Airport, Amsterdam (20-year-old British national) |
| Second admin arrest | August 2024, Ireland (23-year-old Irish national) |
| Cryptocurrency seized | > EUR 8 million |
| Servers seized | Multiple (database / crypto / web), most located in the Netherlands |
| Peak monthly turnover | EUR 12 million (September 2023) |
| Daily ads | ~82,000 |
| Monthly transactions | ~67,000 |
| Estimated admin profits | ~EUR 5 million |
| Dutch-origin transactions | ≥ 14,000, ~EUR 1.7 million |

### Cooperation partners explicitly named

- **Netherlands** — Politie (Team High Tech Crime, National Investigation and Intervention Unit) + OM (Rotterdam prosecution).
- **Ireland** — An Garda Síochána.
- **United Kingdom** — National Crime Agency (NCA).
- **United States** — "U.S. law enforcement" (one tier-2 source specifies DOJ; not uniformly confirmed).

### Cooperation partners NOT confirmed by tier-1

- **Germany / BKA** — appeared in one tier-2 summary (The Record) but is not corroborated by other sources covering the Politie release. Not asserted on the operation page absent direct primary-source confirmation. Per L17 / L19.
- **Eurojust / Europol** — no explicit coordination cited for the original Bohemia/Cannabia takedown. (Operation RapTor, a downstream Europol-coordinated buyer/seller sweep using Bohemia data, is a separate operation announced May 2025.)
